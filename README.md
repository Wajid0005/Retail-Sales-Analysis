### 1. Sales per Gender and Category
Count the number of purchases broken down by both customer gender and product category.

```sql
select 
	gender,
    category,
    count(*) as NUM_OF_PURCHASED
from sql_p1.retail_sales_data
group by 
	gender, 
    category
order by
	gender,
    category;
2. Sales on a Specific Date (Nov 5, 2022)

4. Total Sales by Category
Calculate the total revenue generated by each product category.

SQL

select
	category as CATEGORY,
    sum(total_sale) as TOTAL_SALES
from sql_p1.retail_sales_data
group by category
order by CATEGORY desc;
5. Average Customer Age by Category
Find the average age of customers who purchased products in the 'Beauty' category.

7. Monthly Sales Analysis & Top Sales
Analyze sales trends over time, including average monthly sales and identifying top-performing individual sales.

7.1. Average Sale per Month/Year

SQL

select
	extract(year from sale_date) as sale_year,
    extract(month from sale_date) as sale_month,
    avg(total_sale) as avg_sales
    
from sql_p1.retail_sales_data

group by
	sale_year,
    sale_month
order by
	sale_year,
    sale_month;
7.2. Top 10 Sales Greater than Average Find the top 10 individual sales that were higher than the overall average sale.

SQL

select
	extract(year from sale_date) as YEAR,
    extract(month from sale_date) as MONTH,
    extract(day from sale_date) as DAY,
    total_sale as SALES_GREATER_THAN_AVG
from sql_p1.retail_sales_data
where 
	total_sale > (
		select avg(total_sale)
        from sql_p1.retail_sales_data
        )
order by
	SALES_GREATER_THAN_AVG desc
    limit  10;

10. Sales by Time of Day
Create time-based shifts (Morning, Noon, Evening, Night) and count the number of transactions in each shift.

SQL

select
	case
		when sale_time between '06:00:00' and '11:59:59' then 'MORNING'
        when sale_time between '12:00:00' and '17:59:59' then 'NOON'
        when sale_time between '18:00:00' and '20:59:59' then 'EVENING'
		else 'NIGHT'
	end as sales_shift ,
    count(*) as Transaction
from 
sql_p1.retail_sales_data
group by sales_shift
order by Transaction desc
;
